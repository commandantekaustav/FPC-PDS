Principles of Data Science

A 75-Hour Course Following a Strict 80/20 Hands-On Model

Course Description

This course lays the foundational framework for understanding and practicing data science. Students will journey through the complete data science workflow, from defining a business problem and collecting data to cleaning, analysis, visualization, and predictive modeling. With a curriculum that is 80% practical, the course emphasizes hands-on experience with essential tools like Python, Excel, and Tableau, while also covering the critical ethical considerations and future trends that shape the field.

Key Resources

Core Textbooks & Readings

  • [VanderPlas] Python Data Science Handbook by Jake VanderPlas.
  • [McKinney] Python for Data Analysis by Wes McKinney.
  • [Knaflic] Storytelling with Data by Cole Nussbaumer Knaflic.

Key Platforms

  • [Python] with libraries like Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.
  • [Tableau / Power BI] for dashboarding and business intelligence.
  • [Microsoft Excel] for foundational data analysis and modeling.
